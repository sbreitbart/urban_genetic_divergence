# Subdirectory: CommonGardenExperiment_2019Data

## Folder description

### raw_data

This folder contains files with raw data collected in the field as well as Haversine distances between the city center and each population's sampling location (`Transect_Milkweed_HaversineData_forjoining.csv`).

### clean_data

This folder contains the cleaned versions of the raw data files (i.e., cleaned with scripts in the scripts/cleaning_raw_data/2019_data folder).
